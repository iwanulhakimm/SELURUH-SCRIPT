# ARASUN

ARASUN (Aksara Sunda Augmented Reality) merupakan media pembelajaran berbasis Web Augmented Reality yang dirancang untuk membantu siswa SMP mempelajari Aksara Sunda secara interaktif.

## Fitur
- Pembelajaran Aksara Ngalegena
- Pembelajaran Aksara Swara
- Pembelajaran Angka Sunda
- Pembelajaran Rarangken
- Augmented Reality berbasis Marker
- Game Matching
- Game Susun Kata

## Teknologi
- Unity
- WebGL
- Imagine WebAR
- C#
